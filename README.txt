For the program to run, it has to use a dataset so I use the predict function to run the code.

I used 4 data sets: car.data, nursery.dat, somerville.data, wine.data

I created a command to call the function predict under def predict():

It is predict('datasets/____.data') where in place of the ____ we have to type the dataset we want to run the code

Options are: 1.)predict('datasets/car.data')
	     2.)predict('datasets/nursery.data')
	     3.)predict('datasets/somerville.data')
	     4.)predict('datasets/wine.data')
